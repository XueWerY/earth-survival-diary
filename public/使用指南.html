👣足迹记录

足迹模块用于记录每日的活动行为和用时，按时段（上午/下午/晚上）分组展示。

核心功能

  • 农历日期选择器：默认显示当前周，可展开查看整月日历
  • 添加足迹：记录活动名称、时间段、备注
  • 添加日记：书写每日文字记录
  • 双击快捷编辑：足迹名称和备注支持双击直接编辑

使用方法

  1. 选择日期（默认今天）
  2. 点击"添加足迹"按钮
  3. 填写活动名称、起止时间、备注
  4. 保存后足迹会按时段分组显示

📝笔记

笔记模块支持 Markdown 编辑，左侧显示大纲导航，提供完整的编辑工具栏和实时预览。

核心功能

  • 分类管理：支持自定义分类（颜色、图标）
  • Markdown 编辑器：即时渲染模式，所见即所得，内置工具栏快速插入语法
  • 完整工具栏：支持标题、粗体、斜体、删除线、引用、列表、代码、链接、表格等
  • 大纲导航：左侧面板根据标题层级自动生成大纲，点击跳转
  • 面包屑导航：点击">"下拉快速切换分类和笔记
  • 收藏与快速访问：可将常用视图收藏
  • 排序：按更新时间、创建时间、标题排序

项目文档

项目规范和使用指南已迁移为 HTML 文件：项目规范.html 位于仓库根目录；使用指南.html 位于 public/ 目录随应用打包，可在"我的"页面点击"使用指南"按钮在应用内查看。系统不再自动创建这两篇笔记，老用户已有的笔记原样保留，可继续编辑或删除。

大纲层级

  • 一级标题（#）：最大字号，左侧蓝色边框，加粗显示
  • 二级标题（##）：次大字号，渐进缩进
  • 三级标题（###）：标准字号，更深缩进
  • 四~六级标题：小字号，最深层级缩进
  • 长标题自动截断省略，鼠标悬停显示完整文字

⏱️专注计时

专注模块提供番茄钟和正计时两种模式，帮助你保持专注。

核心功能

  • 番茄钟模式：固定时长倒计时（默认 25 分钟）
  • 正计时模式：自由计时，手动停止
  • 星轨环动画：专注进行时的视觉反馈
  • 常用专注：保存常用专注事项，快速选择
  • 自动生成足迹：专注完成后自动记录到足迹
  • 专注统计：查看专注时长统计

使用方法

  1. 输入专注事项名称（必填）
  2. 可选：添加备注
  3. 点击"开始专注"
  4. 专注完成后自动保存并生成足迹记录

📋清单

清单模块提供智能清单和自定义文件夹管理任务，支持分组、优先级、重复任务等。

核心功能

  • 智能清单：今天、已过期、未来七天自动筛选
  • 自定义文件夹和清单：层层深入管理
  • 分组管理：清单下可创建多个分组
  • 任务属性：日期、时间、优先级、备注、检查事项
  • 重复任务：支持按天/周/月/农历重复
  • 提醒功能：提前提醒（分钟/小时/天）
  • 面包屑导航：下拉快速切换清单
  • 收藏与快速访问：常用视图收藏

数据持久化

清单模块的收藏和快速访问数据持久化到 data/<用户ID>/list/ 目录下。

🚀倒数日

倒数日模块帮你追踪重要日期，系统会自动生成常见节日倒数日。

核心功能

  • 分类管理：生日、纪念日、节日、旅行等
  • 自动节日：系统自动生成重要节日倒数日
  • 倒数/累计：支持倒数（距离还有几天）和累计（已过几天）
  • 置顶：重要倒数日可置顶显示
  • 面包屑导航：下拉快速切换分类

📚课程表

课程表模块按教学周和节次管理课程，支持周视图查看。

核心功能

  • 教学周切换：左右箭头切换周次
  • 节次管理：在"我的"页面配置节次时间
  • 课程属性：名称、地点、教师、颜色、周次
  • 多周次选择：一门课可分布在多个周次
  • 空数组表示全部周：weeks 为空表示该课程在所有周都有

课程提醒卡片

上课时间根据第 1 节开始时间、每节时长、课间休息、午休、晚休和课表节数动态计算。提醒卡片每 5 秒弹出一张（与桌面端一致）。

📊统计

统计模块通过图表展示足迹和专注数据，帮助你了解时间分布。

核心功能

  • 日期范围筛选：自定义统计时间段
  • 足迹统计：按时段、日期统计活动分布
  • 专注统计：专注时长、次数趋势图
  • ECharts 可视化：折线图、饼图、柱状图

🧰工具箱

工具箱汇集了实用小工具和已安装的插件。

核心功能

  • 数据工具：导出数据、导入数据、清理数据
  • 插件列表：查看和使用已安装的插件
  • 工具卡片：点击即可打开使用

🧩插件系统

插件系统支持扩展应用功能，插件位于 src/plugins/ 目录下。

内置插件

  • 文件管理器：含 HTTP 服务器功能
  • 随机数生成器：生成随机数字
  • 提醒查看器：查看所有提醒

插件结构

每个插件包含 index.ts 入口、plugin.json 元数据、tools/ 工具目录。通过工具箱页面可访问插件功能。

👤用户系统

用户系统支持多用户隔离，每个用户的数据独立存储。

核心功能

  • 注册/登录：邮箱 + 密码认证
  • 个人信息：昵称、生日管理
  • 账号安全：修改密码、手机号绑定
  • 桌面端设置：开机自启、关闭行为、窗口分辨率（默认以 1920×1080 打开）
  • 数据隔离：每个用户数据存储在独立目录

"我的"页面工具

"我的"页面"关于"区块提供以下快捷工具按钮：

  • 检查更新：手动检查应用新版本
  • 查看更新日志：弹窗查看版本变更记录
  • 新手引导：重新打开新手引导教程
  • 使用指南：在当前页面内查看本使用指南，原"我的"页面内容会被完全替换（避免内容重叠），页面采用玻璃质感样式与应用星空背景融合，点击右上角关闭按钮可返回"我的"页面

🔄自动更新

Windows 桌面端支持自动更新，通过 electron-updater 实现。

核心功能

  • 自动检查更新：启动时自动检查
  • 手动检查：在"我的"页面点击"检查更新"
  • 更新日志：查看版本变更记录
  • 后台下载：下载完成后提示安装

注意事项

  • CapacitorHttp 请求无默认超时，需同时设置 connectTimeout/readTimeout 并用 Promise.race 加 JS 层超时兜底
  • Android 需通过应用商店或手动安装更新

🧭新手引导

新手引导帮助新用户快速了解各功能模块的使用方法。

核心功能

  • 首次启动自动播放：注册后首次进入应用自动启动引导
  • 分模块引导：足迹、笔记、专注、清单、倒数日、课程表、统计、工具箱、我的
  • 高亮提示：引导步骤高亮目标区域并显示说明
  • 随时重新查看：在"我的"页面可重新打开引导

操作方式

  • 点击"下一步"继续引导
  • 点击"上一步"回看
  • 点击"跳过"结束引导

03数据存储

存储方式

  • Windows 桌面端：本地文件系统（YAML 后端），通过 Express HTTP 服务器读写
  • Android：本地 localStorage 存储
  • 浏览器：通过 API 请求存储到服务端

数据隔离

每个用户的数据存储在独立目录：data/<用户ID>/

用户数据内容

  • 足迹记录（task）
  • 笔记和分类（notes）
  • 专注记录（focusTimer）
  • 清单和任务（list）
  • 倒数日（countdown）
  • 课程表（course）
  • 系统设置和状态（system）
