地球 Online 生存日记 - 使用指南

应用简介

地球 Online 生存日记是一款支持多平台的个人日记应用，支持 Android、Windows、Linux 和鸿蒙系统。

构建安装包

Windows

pnpm electron:build:win

构建产物位于 release/ 目录：Earth-Survival-Diary-Setup-版本号.exe

Linux

pnpm electron:build:linux

构建产物位于 release/ 目录：Earth-Survival-Diary-版本号.deb

Android

通过 Android Studio 打开 android/ 目录，执行 Gradle 构建（Build → Build Bundle(s) / APK(s) → Build APK(s)）。

构建产物位于 android/app/build/outputs/apk/release/：Earth-Survival-Diary-版本号.apk

HarmonyOS（鸿蒙）

pnpm harmony:build
