# 地球 Online 生存日记

一个帮助记录生活足迹、管理任务清单、专注时间的桌面与移动端效率应用，拥有绚丽的星空主题界面、从流星位置延伸至界面边界的拖尾特效，以及 Electron 桌面端的拆分界面功能。新手引导卡片会根据目标元素位置智能判断显示在上方或下方，桌面端和安卓端均能正确显示。安卓端支持全屏显示，应用界面延伸至状态栏和导航栏区域，内容自动适配不被遮挡，同时全局禁用了触控高亮效果，点击元素时不再出现灰色高亮框。安卓端提醒卡片按触发时间依次弹出，同一时刻的提醒每隔 5 秒显示一个，避免同时弹出一堆卡片。写日记弹窗内容输入框的放大编辑弹窗高度为界面高度的 80% 并垂直居中。

## 功能特性

### 👣 足迹记录

- **内联农历日期选择器**：页面顶部始终显示日历面板（宽度为 500px，与卡片展示区一致；屏幕较小时自动缩小到界面宽度的 80%），每个日期下方展示农历（含节气、节日）；收起状态自动展示选中日期所在周；选中和今日日期以加粗彩色突出显示（无背景框）。选择器与下方卡片展示区之间间隔 16px
- 卡片展示区（含重要区域、倒数日、正数日、上午/下午/晚上时段、日记）整体宽度为 500px 水平居中，无背景框无内边距。重要区域集中展示已设为星标的倒数日卡片，标题"⭐ 重要"金色高亮显示。卡片采用单列垂直布局，每行只展示一张卡片，各卡片保持自身自然高度。上午（🌤️ 橙色）、下午（🌞 青色）和晚上（🌙 紫色）时段标题使用不同颜色和 emoji 图标区分，电脑端底部保留 16px 间距
- 滑动交互：下滑展开完整月份视图，上滑收起为单行；左右滑动切换周或月份
- 日期跳转弹窗：三列滚动选择器（年/月/日），支持公历/农历切换，带回到今日按钮
- 快捷记录：日期选择器标题行分为左右两个区域——左侧展示年月标题文本、右侧放置跳转日期、添加足迹和添加日记三个胶囊按钮。在宽屏（宽度≥500px）时按钮显示图标加文字（跳转日期/记录足迹/写日记），在小屏时仅显示图标。选择器与下方标题行间距 16px
- 记录卡片和日记卡片支持双击快捷编辑：双击标题修改名称（输入框自动获得焦点），记录卡片名称前有 📝 图标与日记卡片区分。记录卡片时间行分为"开始时间 - 结束时间"点击弹出滚轮时分选择器（纯文本无背景框，时间文字使用蓝色），双击备注区域可直接编辑备注（输入框自动获得焦点），无备注卡片显示"双击添加备注"提示。每张卡片右侧保留红色圆形删除按钮（无hover和焦点蓝色效果）。删除确认弹窗宽度 300px（屏幕小于300px时占满屏幕），取消和确认按钮采用图标加文字的胶囊样式
- 时间线视图：按日期查看活动记录、清单任务和课程，按结束时间排序
- 记录卡片显示时长标签（共xx小时xx分钟），不足1小时不显示小时数
- 足迹的增删改：事件名称、日期、起止时间、备注
- 新用户自动创建：注册或登录后系统自动创建一条示例足迹记录，事件名称为"这是一个足迹"，开始时间为注册前一分钟，结束时间为当前时间，备注为"这里是备注"
- 时间选择器新建时默认显示当前时间，已有记录时显示该记录保存的时间，选中后加粗并高亮为蓝色
- **公历/农历双历法**：DateScrollPicker 滚轮日期选择器支持公历和农历切换，农历模式下显示中文月份和日期（节气优先），弹窗底部“回到今日”、“取消”、“保存”按钮居中排列
- 清单任务展示：对应日期的清单任务直接显示在足迹页面，保留编辑、移动、删除、检查事项和备注全部功能，已完成任务名称前显示对钩
- 日记卡片（独立 DiaryCard 组件）：开始结束时间都不填时为日记卡片，显示创建时间。支持双击名称和备注进行快捷编辑，右侧显示圆形删除按钮。写日记弹窗标题为"写日记"，标题下方有分隔线，包含名称和内容输入框
- 新用户自动创建：注册或登录后系统自动创建一篇示例日记，名称为"这是一篇日记"，备注为"这里是备注"
- 卡片数据收集系统：切换日期后自动查询倒数日、正数日、清单任务和课程，生成对应卡片
- 课程展示：今日课程直接显示在足迹页面，课程卡片（独立 CourseCard 组件，自带完整卡片背景框样式）采用5行布局——第一行📖图标与课程名称（名称颜色与课程表模块一致）、第二行上课周次/节次/起止时间、第三行上课地点、第四行授课教师、第五行备注，与其它类型卡片清晰区分。课程起止时间与课程表页面保持一致（动态计算，不再使用可能过期的存储数据）
- 记录卡片封装：记录卡片（RecordCard 独立组件）在上午/下午/晚上时段复用，支持双击编辑名称和备注、时间选择器设置起止时间、显示时长标签、红色删除按钮

### ⏱️ 专注计时

- 番茄钟计时与正计时两种模式，切换按钮和开始专注按钮都是图标加文字样式，开始按钮放在输入框下方居中
- 星轨环视觉动画，性能优化后安卓端流畅运行
- 自定义时长：灵活设置专注时间
- 自动记录：专注完成后自动生成足迹
- 常用专注：保存常用事项快速开始
- 取消/完成按钮：计时中显示的取消和完成按钮采用图标加文字样式，与切换按钮风格统一
- 提醒弹窗：番茄钟完成后通过页面上方居中的悬浮弹窗提醒，最新弹窗会叠加在最上方显示，已接入统一提醒调度系统；正计时模式不会触发提醒调度，避免干扰
- 设置统一管理：专注时长等设置已整合到全局用户设置中，避免设置分散存储
- 专注状态指示：桌面端专注计时中，界面右下角会显示当前专注事项和计时信息，方便随时查看进度

### 📋 清单

- **全局导航栏**：顶部（桌面端）或底部（安卓端）显示八个功能模块的导航栏（👣足迹、🧘专注、📋清单、⏳倒数日、📖课程表、📊统计、🧰工具箱、👤我的），桌面端和安卓宽屏（>500px）宽度 500px 水平居中，安卓小屏（≤500px）宽度 80% 自适应，导航项在栏内居中排列；当导航项总宽度超出屏幕时自动变为左对齐并支持左右滑动浏览，与上下内容区间距 16px，图标在上、描述文字在下纵向排列，当前所在模块以白色加粗文字标识（无高亮背景框），点击即可快速切换。切换模块时自动清空前一个模块的状态并交由新模块自行恢复浏览位置，确保页面按钮和内容正确加载。导航项鼠标悬停不显示提示文字
- **拆分界面（桌面端）**：右键全局导航栏弹出菜单，可选择拆分界面或退出拆分模式。拆分后每个面板拥有独立的导航栏，可分别切换不同功能模块。拆分面板之间以竖线分隔，每个面板可独立滚动和交互
- **独立面包屑导航**：清单页面和倒数日页面各有独立的面包屑地址栏，宽度为页面宽度的 80%，水平居中并与顶部导航栏保持间距，格式为"模块图标 > 智能清单/文件夹名 > 清单名 > 分组名"，所有文本段使用对应颜色渲染——智能清单为紫色、文件夹和清单/分组使用其 color 属性、智能清单子项（今天/已过期/未来七天）分别使用绿色/红色/蓝色
- **下拉列表交互**：面包屑中每个 `>` 均可点击弹出下拉列表——弹出对应层级的列表（带颜色标记），点击项即可跳转
- **自适应布局**：桌面端保持 500px 固定宽度，安卓端各页面宽度自适应为屏幕宽度的 80%（最大 500px），提供更好的移动端阅读体验；安卓端清单模块的任务卡片每行只展示一张，卡片阅读更清晰；文件夹卡片、清单卡片、分组卡片、智能清单卡片等网格卡片从每行1列改为每行2列
- **内联农历日期选择器**：日期选择器的年月标签独占第一行居中显示，三个操作按钮（跳转日期、添加足迹、添加日记）以胶囊按钮样式排在第二行居中，清晰易用；安卓小屏幕下自动缩小按钮文字确保在一行完整显示且保持居中，禁用默认触控高亮
- **简化登录注册**：登录和注册页面去除了"导入数据"功能，聚焦核心的注册登录流程；安卓端标题字体缩小，更适合手机屏幕；登录和注册页面之间切换时有平滑的淡入淡出过渡动画；修复了安卓端点击按钮时出现默认触控高亮的问题
- **安卓端兼容性**：修复了清单页面在安卓端因接口地址不匹配导致无法正常操作的 bug；修复了添加任务时数据加载失败的 bug（Capacitor 文件存储函数的返回键名与 API 层不匹配导致任务数据为空）；修复了创建分组时提交数据缺少必要参数导致分组无法创建和报错的问题；面包屑地址栏在路径过长时自动左滑显示末端路径文本；课程表页面宽度调整为屏幕宽度的 100%，各字体大小进一步缩小以更好适配手机屏幕，左侧节次列宽度自适应文本，标题栏不再显示日期数字，课程卡片信息分四行完整显示，点击卡片弹出课程详情弹窗并可从详情进入编辑；课程表各列改为根据屏幕宽度的等宽固定列，行高和午休/晚休区域固定为 64 像素
- **页面操作按钮**：清单和倒数日页面在各自面包屑地址栏右侧显示添加按钮
- **导航状态验证**：恢复浏览位置时严格验证路径是否属于当前模块，防止错误状态导致页面空白
- **调试日志支持**：清单页面及相关模块（usePageNav、missionStore、CountdownList、TaskList、App.vue）已添加完整的 debug 级别日志，方便定位页面渲染和导航状态问题，可通过浏览器控制台查看详细执行流程
- **内容区布局**：使用 `el-scrollbar` 包裹内容，通过 `:deep()` 样式确保滚动区域正确填充父容器高度，`__view` 设置 `min-height: 100%` 保证内容撑满视口，与足迹页面布局一致
- **数据源修复**：清单模块中"清单列表"（taskLists）和"任务列表"（lists）的数据源已正确分离，查找清单和分组信息时不再错误地查询任务数据；包括面包屑导航到清单层级时正确显示分组卡片
- **检查事项优化**：编辑检查事项时如果清空文本内容，该检查事项会自动删除，无需手动操作
- 重复策略：支持不重复、每天、每隔 N 天、每周选择、每月指定日期、每月最后一天、农历等多种重复策略；每月指定日期模式下，如果指定日期在下个月不存在会自动跳过，顺延到有该日期的月份。农历重复策略支持选择农历月（正月~腊月）和农历日（初一~三十下拉选择），任务卡片上显示"每年正月初一"这样的汉字格式，完成任务后系统根据农历月日自动推算下一个公历日期
- 新用户自动创建：注册或登录后系统自动创建一个名为"默认清单"的清单（归属"我的清单"文件夹），包含一个默认分组，分组下自动生成一个示例任务，备注为"这里是备注"，帮助新用户快速上手

### 🚀 倒数日

- 倒计时管理：创建重要事件的目标日期，支持倒数日和正数日两种模式
- 农历日期：支持农历生日、传统节日等
- 分类管理：首页以网格形式展示所有分类卡片（与清单文件夹卡片统一布局和样式），桌面端每行三列、移动端每行两列，80% 宽度水平居中。"全部"分类卡片优先展示，点击可查看所有倒数日和正数日。从其它页面切换回倒数日模块时自动恢复上次查看的分类页面
- 独立面包屑导航：倒数日页面始终显示面包屑地址栏（含首页），宽度80%水平居中，支持下拉列表切换分类，与下方卡片区保持16px间距
- 星标置顶：重要事件优先展示，系统自动生成的倒数日仅生日和使用天数为默认星标
- 删除确认：点击倒数日卡片删除按钮后弹出确认弹窗，提示图标与标题文本水平居中显示，底部取消和确认按钮采用图标加文字的胶囊样式
- 节日倒数日：元旦、春节、元宵、清明、端午、七夕、中元、中秋、重阳、腊八、小年、除夕等二十余个常见节日自动生成，农历节日精确计算，过期自动顺延至明年
- 提醒功能：支持准时提醒和提前提醒（两个单选按钮切换，选中提醒后显示滚轮式提醒时间选择器），已接入统一提醒调度系统；卡片上点击提醒文本弹出仅含提醒设置的编辑窗口
- 重复策略：新增/编辑倒数日时支持设置"不重复"或"重复"（仅倒数日支持，正数日不适用），公历日期按每年重复，农历日期按农历年重复；卡片上点击重复文本可直接切换每年重复/不重复。系统自动创建的节日、生日等默认设为每年重复，修改日期或重复策略后自动重新调度提醒
- 快捷编辑：双击卡片名称区域可直接修改名称（多行文本，输入框自动获得焦点），点击"日历图标+日期文本"弹出滚轮式农历日期选择器修改日期，双击描述区域可添加或编辑描述（输入框自动获得焦点）。卡片右上角操作区新增编辑按钮，点击打开编辑弹窗支持修改所有字段。安卓端卡片按钮始终可见，桌面端鼠标悬浮时显示。页面完整隐藏了滚动条（包括 Element Plus 组件自带的竖向和横向滑块），滚动功能不受影响
- 可复用卡片组件：CountdownCard 独立封装，支持倒数日和正数日两种模式，卡片布局优化为三区域对齐（图标、名字、按钮），"xx天"区域使用深色背景，即将到来卡片背景和名称使用红色突出显示，星标和删除按钮颜色适配卡片风格；正数日卡片不再显示占位区域，布局更简洁
- 足迹页面集成：倒数日/正数日卡片组件同时在倒数日模块和足迹页面中使用，数据完全同步；足迹页面使用完整卡片模式，支持星标、删除、编辑名称、描述、日期等完整交互操作；正数日仅星标后才会在足迹页面展示，未星标正数日不在足迹页面显示；取消星标后渲染稳定，卡片不会重复显示或消失
- 新增/编辑弹窗：宽度固定300px，标题居中无关闭图标，起始日期区域130px宽、分类区域90px宽，底部按钮为图标胶囊样式。正数日模式下不显示重复策略和提醒设置选项
- navPath 导航：使用共享全局导航栏，支持分类切换下拉列表

### 📚 课程表

- 周视图网格：按节次和星期划分的网格视图，整周课程一目了然。星期以独立标签形式展示在网格上方，午休和晚休区域同样以独立标签展示在对应网格段之间
- 网格合并：跨越多个节次的课程自动合并显示，全周课程全选时合为一整行（含节次标签）
- 单击操作：单击空白网格添加课程，单击课程卡片编辑课程，悬停空白处显示加号图标（添加时自动选中点击位置对应的上课日，编辑时可切换到其他天）
- 课程卡片：显示课程名、地点、教师、备注和节次起止时间，文字支持多行显示，全周卡片水平垂直居中。安卓端卡片内容分四行完整显示，不再截断省略
- 上课日选取：添加或编辑课程时通过单选按钮选取上课日（周一~周日），每次只能选一天，操作更简单直接
- 午休/晚休：上午最后一节与下午第一节之间显示午休过渡区域，下午最后一节与晚上第一节之间显示晚休过渡区域
- 课表节数设置：在课程表页面左上角设置按钮打开的弹窗中可分别设置上午、下午和晚上的节数（滚轮选择器，每列范围1-10，显示3个待选项）
- 午休/晚休时长设置：在课程表设置弹窗中可设置午休和晚休的时长（分钟数）
- 课间休息时长支持统一和自由两种模式：统一模式所有课间使用相同分钟数，自由模式可为每两节课之间分别设置不同休息时长。设置弹窗中每项设置卡片采用三行排列（设置名、说明、输入区），课间休息时长的模式切换单独成行，布局清晰整洁
- 智能节次计算：所有节次的起止时间完全根据设置中的第1节开始时间、每节时长、课间休息时长（统一/自由）、课表节数（上午/下午/晚上）、午休时长和晚休时长自动动态生成，无需手动编辑节次；在课程表设置弹窗中修改任意设置后，课程表页面的节次信息立即自动同步更新
- 非本周课程显示：开启"显示非本周课程"后，课程卡片上会显示对应的周次信息
- 智能数据存储：课程使用节次编号数组存储，修改设置后网格自动更新，无需手动调整
- 40色颜色选择器：与清单文件夹颜色选择器同款，支持自定义颜色输入
- 删除课程和节次使用自定义确认弹窗，显示在编辑弹窗上方清晰可见
- 周次切换：左右箭头切换周次，周次和日期范围居中显示于导航栏下方
- 新用户自动创建：注册或登录后系统自动创建一个名为"这是一门课程"的示例课程，上课时间为注册当天对应的周几第1节，备注为"这里是备注"，方便新用户了解课程表功能

### 📊 统计

- 五合一统计面板：在一个页面同时查看足迹、专注、清单、倒数日和课程表五大模块的使用情况
- 滚轮农历日期选择器：支持公历和农历自由切换选择开始和结束日期，默认统计从注册当天到今天
- 图标胶囊重置按钮：一键恢复默认统计范围
- 安卓端页面宽度自适应为屏幕宽度的100%，与左右边界保留间距；统计范围区域拆分为两行，第一行"统计范围"和重置按钮居中显示，第二行日期选择器居中显示
- 足迹统计：日记数和记录数分开计算，日均日记数和日均记录数，总时长；每日日记数趋势柱状图和每日记录数趋势柱状图分别展示

## 数据存储

**桌面版（Windows）**：所有数据存储在 `%APPDATA%/earth-survival-diary/data/` 目录中，通过内嵌 Express 服务器读写 JSON 文件。用户数据通过 userId 隔离。更新或重装不会影响已有数据。卸载时可选择是否删除。桌面窗口默认大小为 1280×720。

**安卓版**：与桌面版使用完全相同的路径化 JSON 文件结构存储数据，通过 Capacitor Filesystem 插件直接读写设备本地的 JSON 文件，无需联网，所有操作在设备端完成。卸载应用会清除数据。注销账号时，优先使用系统原生递归删除，若失败则自动降级为逐文件删除，确保所有数据被彻底清除。

日志文件存储在 `%APPDATA%/earth-survival-diary/logs/` 目录中，按日轮转（app-YYYY-MM-DD.log）。安卓端同样支持日志文件写入，采用 1 秒防抖合并写入策略（每次写入完整内存缓冲区，首次写入时自动读取磁盘历史日志合并写入，确保重启后不丢失），减少 Capacitor 原生调用频率

### 用户数据内容

每个用户的数据包含：

- 个人资料：昵称、生日
- 足迹记录：活动名称、日期、起止时间、备注
- 任务清单：清单、分组、任务项、检查事项
- 倒数日：事件、目标日期、分类、星标
- 课程表：课程安排、地点、教师、周次
- 专注记录：常用专注事项
- 设置项：专注时长、学期信息、第1节开始时间、每节时长、课间休息时长、课表节数设置（上午/下午/晚上节数）、周末显示开关、非本周课程显示开关等应用配置

## 构建 Windows 应用

### 环境要求

- Node.js >= 18
- pnpm

### 安装依赖

```
pnpm install
```

### 打包

```
# 构建 Windows 安装包
pnpm electron:build:win
```

构建完成后，安装包输出在 `release/` 目录下。

## 构建 Android 应用

### 环境要求

- Node.js >= 18
- pnpm
- Android Studio（用于编译 APK）

### 开发调试

```
# 构建 Web 资源并同步到 Android 项目
pnpm build && npx cap sync

# 在 Android Studio 中打开项目
npx cap open android
```

在 Android Studio 中连接设备或启动模拟器，点击 Run 即可。

### 构建 APK

在 Android Studio 中选择 Build → Build Bundle(s) / APK(s) → Build APK(s)。

构建完成后，APK 输出在 `android/app/build/outputs/apk/` 目录下。
